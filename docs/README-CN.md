个人的 Vim/Neovim 配置文件，适用于 Linux、Windows 系统。

## 特性

+ 遵循模块化原则。
+ 支持主流语言的代码查错、代码补全、代码格式化。

## 使用

如要你要使用这份配置文件，请按照如下的步骤操作。

+ ``git clone --depth=1 https://github.com/aj-ash/vimrc``
+ ``cd vimrc/tools``
+ 根据你所使用的系统，运行 Restore 脚本。

当然，这份 Vim/Neovim 的配置文件可能更符合我的使用习惯。
